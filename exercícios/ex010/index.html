Links Externos

É possivel acessar qualquer site.

Nesse caso a tag <a> faz o site aberto atráves do link se sobrepôr a minha página, esse é o menos recomendado a se usar, a menos que seja um link para outra sessão dentro do mesmo site.

Links Externos em uma nova guia

Para acessar qualquer link criando uma nova guia no navegador, usa-se "target=_blank rel=external", dentro da tag <a>.

Links Internos

São links usados para navegar por tópicos dentro do meu servidor, não tem diferença entre usar ou não "target=_blank rel=external", dentro da tag <a>, mas recomenda-se usar "rel=next", ou "rel=prev", para "dizer", ao mecânismo do navegador qual a pagina anterior, ou qual a próxima página.
